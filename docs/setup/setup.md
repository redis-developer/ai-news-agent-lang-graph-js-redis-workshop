# Setup Guide

## Finding Your Code

In VS Code, you'll see a folder structure on the left:

```
code/
└── web/
    ├── package.json
    ├── vite.config.js
    ├── index.html
    ├── style.css
    └── main.js
```

All your work will be in the `web` folder.

## Understanding the Files

| File | Purpose |
|------|---------|
| `index.html` | The main HTML structure |
| `style.css` | All your styles and colors |
| `main.js` | JavaScript code |
| `package.json` | Project dependencies |
| `vite.config.js` | Vite configuration |

## Viewing Your App

1. Open the "App" panel in the workbench
2. You should see "Hello World! 👋"
3. Keep this tab open - it will auto-update as you code!

## Making Your First Edit

Let's test that everything works:

1. In VS Code, open `web/index.html`
2. Find the line with "Hello World! 👋"
3. Change it to "Hello Workshop! 🚀"
4. Save the file (Ctrl+S or Cmd+S)
5. Watch your app tab update instantly!

If you see the change, you're ready to go! 🎉

## Tips

- **Save often** - Changes only appear after saving
- **Use both screens** - VS Code on one side, app on the other
- **Check the console** - Press F12 in your app to see errors
- **Don't edit other files** - Stick to the `web` folder for now
