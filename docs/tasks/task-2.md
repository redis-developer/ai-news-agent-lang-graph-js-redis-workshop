# Task 2: Add Styling

## Goal

Learn how to modify CSS to change colors, fonts, and layout.

## Instructions

### Step 1: Open the CSS File

1. In VS Code, open `web/style.css`
2. Notice how the styles are organized

### Step 2: Change the Background

Find this line:
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

Try changing the colors! Here are some options:

**Blue to Green:**
```css
background: linear-gradient(135deg, #11998e 0%, #38ef7d 100%);
```

**Orange to Pink:**
```css
background: linear-gradient(135deg, #f093fb 0%, #f5576c 100%);
```

**Or use a solid color:**
```css
background: #2c3e50;
```

### Step 3: Change the Heading Color

Find the `h1` section and change the color:

```css
h1 {
    color: #ff6b6b;  /* Try different colors! */
    font-size: 3rem;
    margin-bottom: 1rem;
}
```

### Step 4: Experiment!

Try changing:
- Font sizes (`font-size`)
- Padding and margins
- Border radius (`border-radius`)
- Box shadow

## Challenge

Can you make the container have a colored border? Try adding this to `.container`:

```css
border: 5px solid #667eea;
```

## What You Learned

✅ How to modify CSS  
✅ How to change colors and gradients  
✅ How to style elements
