# Task 1: Hello World

## Goal

Customize the welcome message and learn the basic workflow.

## Instructions

### Step 1: Open the HTML File

1. In VS Code, navigate to `web/index.html`
2. Look at the structure - notice the `<h1>` and `<p>` tags

### Step 2: Change the Heading

Find this line:
```html
<h1>Hello World! 👋</h1>
```

Change it to something personal:
```html
<h1>Welcome to [Your Name]'s Workshop! 🚀</h1>
```

### Step 3: Update the Paragraphs

Modify the paragraph text to say something different. Be creative!

### Step 4: Save and View

1. Save the file (Ctrl+S or Cmd+S)
2. Check the "App" panel to see your changes.
3. Your changes should appear instantly!

## Challenge

Can you add a new paragraph? Try adding this after the existing paragraphs:

```html
<p>This is my first edit in the workshop!</p>
```

## What You Learned

✅ How to edit HTML files  
✅ How to save changes  
✅ How hot reload works with Vite
