## Common Tasks

### Adding a New HTML Element

```html
<div class="my-element">
    <p>Content here</p>
</div>
```

### Styling an Element

```css
.my-element {
    background: #f0f0f0;
    padding: 1rem;
    border-radius: 8px;
}
```

### Selecting Elements in JavaScript

```javascript
// By ID
const element = document.getElementById('myId');

// By class
const elements = document.querySelectorAll('.myClass');

// By tag
const paragraphs = document.querySelectorAll('p');
```

## Troubleshooting

### Changes Not Appearing?

1. Make sure you saved the file (Ctrl+S / Cmd+S)
2. Check the browser console for errors (F12)
3. Try refreshing the page manually

### Syntax Errors?

- Check for missing closing tags in HTML
- Check for missing semicolons in JavaScript
- Check for missing closing braces `}` in CSS
