# Workshop Slides

Presentation slides for the AI News Agent Workshop, built with [MARP](https://marp.app/).

## Setup

```bash
npm install
```

## Usage

**Live preview with hot reload:**

```bash
npm run serve
```

Opens a local server that watches for changes and refreshes automatically. Great for authoring.

**Export to HTML:**

```bash
npm run build:html
```

**Export to PDF:**

```bash
npm run build:pdf
```

## Presenting

1. Run `npm run build:html` to generate `deck.html`
2. Open `deck.html` in your browser
3. Press **F11** to go fullscreen
4. Use arrow keys to navigate slides
5. Press **P** to open presenter mode (speaker notes + next slide preview)
